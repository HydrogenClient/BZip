Usage


# Compress a folder or file
bzip c "input_folder_or_file" "output.bzip"

# Decompress a .bzip archive
bzip d "archive.bzip"

Features

  • Compress files and folders into .bzip archives
  • Decompress .bzip files easily
  • Installer adds BZip to system PATH automatically
  • Small and fast
